# CommonView

## Gradle
[![](https://jitpack.io/v/caohao9808/CommonView.svg)](https://jitpack.io/#caohao9808/CommonView)
 ```
implementation 'com.github.caohao9808:CommonView:Tag'
 ```
 jitpack还要求在工程根目录的`build.gradle`中添加如下：
 ```
 allprojects {
     repositories {
         ...
         maven { url 'https://jitpack.io' }
     }
 }
 ```
